# CMSC 335: final project

**Submitted by:** Mariia Honcharenko (mhonchar)

**Group Members:** Mariia Honcharenko (mhonchar)

**App Description:** A web app built with Node.js, Express.js, and MongoDB. It shows users a random fact from an external API in the language they choose. Users can save facts to a Favorites list stored in MongoDB and view or delete their saved facts on the Favorites page.

**APIs Used:** Useless Facts API ([https://uselessfacts.jsph.pl/](https://uselessfacts.jsph.pl/))
